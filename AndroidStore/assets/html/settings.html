Settings

The Settings screen provides the following options:

  • Region: Select the closest locale you want to use to find items. The default is United States, which is generally recommended for most items. There may be some cases where you'd want to change this. For instance, if you're looking for a French copy of Breton, you might find more accurate results by setting the locale to "France." If you're scanning a British DVD like This is England, set the locale to "United Kingdom." Region affects everything: adding manually, scanning barcodes, finding items online, etc.
  • Calendar: Select the Google Calendar you want to use when adding Loan events. This option is only available if you're using a Google Calendar.
  • Import: Select this to import a list of items from files on your SD card.
  • Export: Select this to export a list of items as a tab-delimited text file to your SD card.
  • Send To Google Docs: Uploads your collection to the Google Docs website as a spreadsheet. Note that you shouldn't rename the spreadsheet after it's uploaded, or you won't be able to import it back into your device. Your collection must be under 1 MB for this to work.
  • Restore from Google Docs: Retrieves a previously uploaded collection from Google Docs, and begins to import it into Shelves. If any of the content was edited, those edits are reflected in the app collection.
  • Delete Everything: Select this to wipe out your entire Shelves collection. You can also use this to restore your collection if something unexpectedly breaks. First, export your items for Shelves. Then, delete everything. Finally, import from Shelves, using the saved export file you just created.
